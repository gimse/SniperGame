# Sniper Game

A creative sniper game. Control the player with "arrow keys", jump with "s", and shoot with "space".

## Screencapture

![Demo image](demo.gif)

## Getting Started

Open this [link](https://rawgit.com/gimse/SniperGame/master/run_game.html). You can also download the repository and open run_game.html in you browser.

### Prerequisites

Flash Professional to edit.
